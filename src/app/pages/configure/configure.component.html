Bittrex API settings
You cannot access any part of the application before entering your api settings.

Your keys will be stored in local storage, therefore, you can generate a new key for each device or re-use the same key on all your devices. You can generate a new api key in your Bittrex settings.

Make sure your connection is secure because your key will be submitted to the server to make a request to bittrex.
